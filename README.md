# PoE Bot

Project organized according to the **DOE Architecture** (Directive, Orchestration, Execution).

## Structure

- `directives/`: Standard Operating Procedures (SOPs) in Markdown.
- `execution/`: Deterministic Python scripts for automation.
- `.tmp/`: Intermediate data files (git-ignored).
- `.venv/`: Mandatory Python virtual environment.

## Getting Started

1. Activate the virtual environment: `.\.venv\Scripts\Activate.ps1`
2. Define directives in `directives/`.
3. Implement execution scripts in `execution/`.
